적용 내용
- 전체 기능 JS/Firebase 로직 유지
- style.css 중심으로 디자인만 변경
- 굵은 선 2px → 대부분 1px로 완화
- 폰트 굵기 800~950 → 400~650 중심으로 조정
- 배경 패턴/강한 그림자 제거
- 좌측 브랜드 로고 마크 추가: CSS pseudo logo 방식
- 라임/퍼플/핑크 포인트는 유지하되 더 미니멀하게 정리

사용 방법
1. 기존 프로젝트 폴더에 이 ZIP 파일을 풀어주세요.
2. 기존 firebase.js, login.html 등 이번 ZIP에 없는 파일은 기존 파일을 그대로 유지하세요.
3. style.css를 덮어씌우면 전체 페이지에 새로운 UI가 적용됩니다.
